# ChangeTheHash
Change the hash so you  can save it like something else.

某些應用（如騰訊qq）會在你上傳文件之後檢查文件hash，以分辨其是否為違規文件。通過改變哈希值的方法，可以使得原本不允許上傳的文件變得可以上傳。

Some software check your file's hash before uploading and block your upload if it finds your file as 'prohibited'. To change the file hash can 
bypass this michanism for now.

軟件通過直接像文件後端增加無效字符的方式改變文件哈希值。這樣做不會讓文件變得不可用（尤其對視頻來説），同時可以改變文件哈希值。

後續版本可能會提供用戶手動輸入混淆字符（串）的功能。
